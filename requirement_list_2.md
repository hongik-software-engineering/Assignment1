# requirement list

---

| No. | Requirement | Use Case |
| --- | --- | --- |
| 1 | 시스템은 회원과 관리자가 로그인 화면에서 ID와 비밀번호를 입력할 수 있는 입력 필드를 제공해야 한다. | 로그인 |
| 2 | 시스템은 로그인 버튼 클릭 시 입력된 ID/비밀번호를 검증하고 인증에 성공하면 해당 계정으로 접속을 허용해야 한다. | 로그인 |
| 3 | 시스템은 인증 실패 시 사용자에게 실패 메시지를 표시해야 한다. | 로그인 |
| 4 | 시스템은 로그인된 사용자가 로그아웃 버튼을 클릭하면 세션을 종료해야 한다. | 로그아웃 |
| 5 | 시스템은 관리자에게만 설문 응답 통계 조회 기능을 제공해야 한다. | 통계 |
| 6 | 시스템은 통계 조회 시 검색 단위를 최근 1주일 / 1개월 / 1년 중 선택할 수 있는 옵션을 제공해야 한다. | 통계 |
| 7 | 시스템은 선택된 기간을 기준으로 전체 설문 수를 산출하여 출력해야 한다. | 통계 |
| 8 | 시스템은 선택된 기간을 기준으로 종료된 설문 수를 산출하여 출력해야 한다. | 통계 |
| 9 | 시스템은 선택된 기간을 기준으로 총 응답 수를 산출하여 출력해야 한다. | 통계 |
