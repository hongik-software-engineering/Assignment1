# use case description

---

## 회원 가입

| Step | Actor Action | System Response |
| --- | --- | --- |
| 1 |  | 시스템은 로그인 화면을 표시한다. |
| 2 | 사용자는 로그인 화면에서 회원 가입 버튼을 선택한다. |  |
| 3 |  | 시스템은 회원 가입 화면을 표시한다. |
| 4 | 사용자는 ID를 입력한다. |  |
| 5 |  | 시스템은 입력된 ID를 화면에 표시한다. |
| 6 | 사용자는 비밀번호를 입력한다. |  |
| 7 |  | 시스템은 입력된 비밀번호를 화면에 표시한다. |
| 8 | 사용자는 이름을 입력한다. |  |
| 9 |  | 시스템은 입력된 이름을 화면에 표시한다. |
| 10 | 사용자는 전화번호를 입력한다. |  |
| 11 |  | 시스템은 입력된 전화번호를 화면에 표시한다. |
| 12 | 사용자는 이메일을 입력한다. |  |
| 13 |  | 시스템은 입력된 이메일을 화면에 표시한다. |
| 14 | 사용자는 가입 완료 버튼을 선택한다. |  |
| 15 |  | 시스템은 필수 입력 정보가 모두 입력되었는지 확인한다. |
| 16 |  | 시스템은 회원 가입 완료 메시지를 표시한다. |

## 회원 탈퇴

| Step | Actor Action | System Response |
| --- | --- | --- |
| 1 |  | 마이페이지 화면에서 "회원 탈퇴" 메뉴를 표시한다. |
| 2 | 회원이 "회원 탈퇴" 메뉴를 선택한다. |  |
| 3 |  | 비밀번호 입력란과 탈퇴 시 회원 정보 및 응답 데이터가 모두 삭제된다는 안내 메시지를 포함한 회원 탈퇴 화면을 표시한다. |
| 4 | 회원이 비밀번호를 입력한다. |  |
| 5 | 회원이 "탈퇴하기" 버튼을 눌러 탈퇴를 확정한다. |  |
| 6 |  | 입력한 비밀번호가 일치하는지 확인한다. |
| 7 |  | 회원 정보(ID, 비밀번호, 이름, 전화번호, 이메일 등)와 해당 회원이 제출한 설문 응답 데이터를 모두 삭제한다. |
| 8 |  | 해당 ID의 모든 이용 권한을 회수하고 자동으로 로그아웃 처리한다. |
| 9 |  | 탈퇴가 완료되었음을 알리는 메시지를 표시하고 메인 화면으로 이동한다. |

**Alternative Courses**

- Step 6. 입력한 비밀번호가 일치하지 않는 경우, 오류 메시지를 표시하고 동일 화면에서 비밀번호를 다시 입력받는다.
- Step 5. 회원이 "취소" 버튼을 누르면 탈퇴 절차를 중단하고 마이페이지로 돌아간다.

## 로그인

| Step | Actor Action | System Response |
| --- | --- | --- |
| 1 |  | 로그인 창을 표시한다. |
| 2 | 관리자나 회원은 아이디와 비밀번호를 입력하고 로그인 버튼을 누른다. |  |
| 3 |  | 인증 성공 시, 메인 화면으로 전환하며 ‘OO님’가 뜬 메인 화면으로 전환된다. |

Alternative Courses

Steps 3-1. 아이디 또는 비밀번호가 유효하지 않은 경우,
오류 메시지를 표시하고 로그인 화면을 유지한다.

## 로그아웃

| Actor Action | System Response |
| --- | --- |
|  |   1. 모든 화면 상단바에 로그아웃 버튼이 있다. |
| 2. 관리자나 회원은 로그아웃 버튼을 누른다. |  |
|  | 3. 로그아웃 여부를 확인하는 메시지를 표시한다. |
| 4. 관리자나 회원은 확인 버튼을 누른다. |  |
|  | 5. 로그아웃되었다는 확인 메시지와 함께 로그인 가능한 메인 화면으로 전환된다. |

## 설문 팝업 조회

| Step | Actor Action | System Response |
| --- | --- | --- |
| 1 |  | 시스템은 설문 리스트 조회 화면을 표시한다. |
| 2 | 관리자는 설문 리스트 조회 화면에서 상세 내용을 확인할 설문 항목을 선택한다. |  |
| 3 |  | 시스템은 선택된 설문의 상세 팝업 창을 표시한다. |
| 4 |  | 시스템은 팝업 창에 설문 제목, 설명, 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 표시한다. |
| 5 | 관리자는 설문 상세 내용을 확인한다. |  |
| 6 | 관리자는 닫기 버튼을 선택한다. |  |
| 7 |  | 시스템은 팝업 창을 닫고 설문 리스트 조회 화면을 표시한다. |

## 설문 검색

| Step | Actor Action | System Response |
| --- | --- | --- |
| 1 |  | 설문 검색 화면을 표시한다 (진행 상태 선택 옵션과 키워드 입력창 포함). |
| 2 | 회원이 검색 대상 설문의 진행 상태(진행 중 / 진행 예정 / 종료) 중 하나를 선택한다. |  |
| 3 | 회원이 검색 키워드를 입력하고 검색 버튼을 누른다. |  |
| 4 |  | 입력한 키워드와 선택한 진행 상태 조건에 부합하는 설문 리스트를 출력한다. |
| 5 | 회원이 검색 결과 리스트에서 특정 설문을 선택한다. |  |

**Alternative Courses**

- Step 4. 조건에 맞는 설문이 없는 경우, 결과 영역에 빈 리스트 또는 "검색 결과가 없습니다" 메시지를 표시한다.
- Step 2~3. 회원이 진행 상태만 선택하고 키워드를 입력하지 않은 경우, 해당 진행 상태의 전체 설문 리스트를 출력한다.

## 설문 상세정보 조회

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 검색된 설문 리스트 화면에서 특정 설문을 선택한다. |  |
| 2 |  | 새로운 화면에서 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 출력한다. |
| 3 |  | 현재 진행 중인 설문인 경우 응답 버튼을 활성화한다. |

## 설문 응답

| **Step** | **User Action** | **System Response** |
| --- | --- | --- |
| 1 | 회원은 활성화된 응답 버튼을 누른다. |  |
| 2 |  | 시스템은 응답 화면으로 이동한다. |
| 3 | 회원은 설문 문항에 응답을 입력한다. |  |
| 4 | 회원은 응답 버튼을 누른다. |  |
| 5 |  | 시스템은 응답을 저장하고 완료 처리한다. |

## 응답한 설문 조회

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 응답한 설문 조회 화면으로 이동한다. |  |
| 2 |  | 시스템은 해당 회원이 응답한 모든 설문 정보를 한 화면에 출력한다. |
| 3 |  | 시스템은 각 설문에 대해 설문 제목, 응답 내용, 응답 일시를 출력한다. |

## 응답 수정

| **Step** | **User Action** | **System Response** |
| --- | --- | --- |
| 1 |  | 시스템은 응답한 설문 조회 화면을 표시한다. |
| 2 | 회원은 응답한 설문 조회 화면에서 수정할 설문의 응답 수정을 선택한다. |  |
| 3 |  | 시스템은 응답 수정 화면으로 이동하며 가장 최근에 수정한 날짜를 출력한다. |
| 4 | 회원은 응답 내용을 수정한다. |  |
| 5 | 회원은 수정 버튼을 누른다. |  |
| 6 |  | 시스템은 수정된 응답을 저장한다. |

**Alternative Courses**

- Step 2. 설문이 종료된 경우 수정 버튼이 비활성화되어 응답 수정이 불가하다.

## **응답 취소**

| **Step** | **User Action** | **System Response** |
| --- | --- | --- |
| 1 |  | 시스템은 응답한 설문 조회 화면을 표시한다. |
| 2 | 회원은 응답한 설문 조회 화면에서 취소할 설문의 응답 취소를 선택한다 |  |
| 3 |  | 시스템은 취소 확인 메시지를 표시한다 |
| 4 | 회원은 확인 버튼을 누른다 |  |
| 5 |  | 시스템은 해당 응답을 삭제한다 |
| 6 |  | 시스템은 갱신된 응답한 설문 조회 화면을 표시한다 |

**Alternative Courses**

- Step2. 설문이 종료된 경우 응답 취소가 불가하다.

## 통계 기능

| Actor Action | System Response |
| --- | --- |
|  |   1. 설문 응답 통계 정보를 조회할 수 있는 화면을 표시한다. |
| 2. 오른쪽에 있는 드롭다운 버튼을 누른다. |  |
|  | 3. 드롭다운 메뉴에 1주일, 1개월, 1년 옵션을 표시한다. |
| 4. 1주일, 1개월, 1년 중 원하는 기간을 선택한다. |  |
|  | 5. 선택한 기간 동안의 전체 설문 수, 종료된 설문 수, 총 응답 수를 출력한다. |
