## 회원가입, 설문 조회/등록/삭제 Use Case Description
| Use Case | Description |
|---|---|
| 회원 가입 | 시스템을 이용하려는 사용자는 회원 가입을 해야한다. 회원 가입 시 ID, 비밀번호, 이름, 전화번호, 이메일을 입력한다. 회원 가입이 완료되면 사용자는 시스템에 로그인하여 시스템을 이용할 수 있다. |
| 설문 등록 | 관리자는 새로운 설문을 등록할 수 있다. 설문 등록 시 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각 등을 입력한다. 등록이 완료된 설문은 설문 리스트에서 조회할 수 있다. |
| 설문 리스트 조회 | 관리자는 등록된 모든 설문 리스트를 조회할 수 있다. 설문 리스트에는 관리자가 등록한 설문 항목들이 표시된다. 관리자는 설문 리스트 조회 화면에서 특정 설문을 선택하여 상세 내용을 확인하거나 삭제할 수 있다. |
| 설문 상세 팝업 조회 | 관리자는 설문 리스트 조회 화면에서 원하는 설문 항목을 선택하여 상세 내용을 팝업 창으로 조회할 수 있다. 팝업 창에는 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각이 표시된다. |
| 설문 삭제 | 관리자는 설문 리스트 조회 화면에서 특정 설문 항목을 선택하여 삭제할 수 있다. 삭제를 선택하면 시스템은 삭제 확인 메시지를 표시하고, 관리자가 확인하면 해당 설문이 삭제된다. 삭제 후 갱신된 설문 리스트가 표시된다. |


### 회원가입
| Step | User Action | System Response |
|---:|---|---|
| 1 | 사용자는 로그인 화면에서 회원 가입 버튼을 선택한다. | 시스템은 회원 가입 화면을 표시한다. |
| 2 | 사용자는 ID를 입력한다. | 시스템은 입력된 ID를 화면에 표시한다. |
| 3 | 사용자는 비밀번호를 입력한다. | 시스템은 입력된 비밀번호를 화면에 표시한다. |
| 4 | 사용자는 이름을 입력한다. | 시스템은 입력된 이름을 화면에 표시한다. |
| 5 | 사용자는 전화번호를 입력한다. | 시스템은 입력된 전화번호를 화면에 표시한다. |
| 6 | 사용자는 이메일을 입력한다. | 시스템은 입력된 이메일을 화면에 표시한다. |
| 7 | 사용자는 가입 완료 버튼을 선택한다. | 시스템은 필수 입력 정보가 모두 입력되었는지 확인한다. |
| 8 |  | 시스템은 회원 가입 완료 메시지를 표시한다. |

### 설문등록
| Step | User Action | System Response |
|---:|---|---|
| 1 | 관리자는 관리자 메인 화면에서 설문 등록 버튼을 선택한다. | 시스템은 설문 등록 화면을 표시한다. |
| 2 | 관리자는 설문 제목을 입력한다. | 시스템은 입력된 설문 제목을 화면에 표시한다. |
| 3 | 관리자는 설문 설명을 입력한다. | 시스템은 입력된 설문 설명을 화면에 표시한다. |
| 4 | 관리자는 설문 문항을 입력한다. | 시스템은 입력된 설문 문항을 화면에 표시한다. |
| 5 | 관리자는 응답 항목을 입력한다. | 시스템은 입력된 응답 항목을 화면에 표시한다. |
| 6 | 관리자는 설문 시작 시각을 입력한다. | 시스템은 입력된 설문 시작 시각을 화면에 표시한다. |
| 7 | 관리자는 설문 마감 시각을 입력한다. | 시스템은 입력된 설문 마감 시각을 화면에 표시한다. |
| 8 | 관리자는 등록 버튼을 선택한다. | 시스템은 필수 설문 정보가 모두 입력되었는지 확인한다. |
| 9 |  | 시스템은 설문 등록 완료 메시지를 표시한다. |

### 설문조회
| Step | User Action | System Response |
|---:|---|---|
| 1 | 관리자는 관리자 메인 화면에서 설문 리스트 조회 메뉴를 선택한다. | 시스템은 설문 리스트 조회 화면을 표시한다. |
| 2 |  | 시스템은 등록된 모든 설문 리스트를 표시한다. |
| 3 | 관리자는 설문 리스트를 확인한다. |  |


### 설문삭제
| Step | User Action | System Response |
|---:|---|---|
| 1 | 관리자는 설문 리스트 조회 화면에서 삭제할 설문 항목을 선택한다. | 시스템은 선택된 설문 항목을 표시한다. |
| 2 | 관리자는 삭제 버튼을 선택한다. | 시스템은 삭제 확인 메시지를 표시한다. |
| 3 | 관리자는 확인 버튼을 선택한다. | 시스템은 설문 삭제 완료 메시지를 표시한다. |
| 4 |  | 시스템은 갱신된 설문 리스트를 표시한다. |


### 설문팝업조회
| Step | User Action | System Response |
|---:|---|---|
| 1 | 관리자는 설문 리스트 조회 화면에서 상세 내용을 확인할 설문 항목을 선택한다. | 시스템은 선택된 설문의 상세 팝업 창을 표시한다. |
| 2 |  | 시스템은 팝업 창에 설문 제목, 설명, 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 표시한다. |
| 3 | 관리자는 설문 상세 내용을 확인한다. |  |
| 4 | 관리자는 닫기 버튼을 선택한다. | 시스템은 팝업 창을 닫고 설문 리스트 조회 화면을 표시한다. |
