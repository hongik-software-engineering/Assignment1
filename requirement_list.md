# Requirement List

| **No.** | **Requirement** | **Use Case(s)** |
| --- | --- | --- |
| 1 | 사용자는 ID, 비밀번호, 이름, 전화번호, 이메일을 입력하여 회원 가입할 수 있다 | 회원 가입 |
| 2 | 회원은 시스템에서 언제든 탈퇴할 수 있도록 한다 | 회원 탈퇴 |
| 3 | 탈퇴 전에 본인 확인을 위해 비밀번호를 재입력하도록 한다 | 회원 탈퇴 |
| 4 | 탈퇴 전에 최종 확인 단계를 거치도록 한다 | 회원 탈퇴 |
| 5 | 탈퇴 시 시스템은 회원 정보(ID, 비밀번호, 이름, 전화번호, 이메일 등)를 삭제한다 | 회원 탈퇴 |
| 6 | 탈퇴 시 회원이 제출한 설문 응답 데이터를 삭제한다 | 회원 탈퇴 |
| 7 | 탈퇴 후에는 해당 ID로 시스템 이용이 불가능하도록 모든 이용 권한을 회수한다 | 회원 탈퇴 |
| 8 | 탈퇴 처리가 완료되면 자동으로 로그아웃 처리한다 | 회원 탈퇴 |
| 9 | 탈퇴 완료 메시지를 회원에게 표시한다 | 회원 탈퇴 |
| **10 | 탈퇴는 로그인 된 회원만 수행할 수 있도록 제한한다 | 로그인 된 회원이 수행하는 모든 use case(공통 요구사항) |
| 11 | 시스템은 회원과 관리자가 로그인 화면에서 ID와 비밀번호를 입력할 수 있는 입력 필드를 제공해야 한다 | 로그인 |
| 12 | 시스템은 로그인 버튼 클릭 시 입력된 ID/비밀번호를 검증하고 인증에 성공하면 해당 계정으로 접속을 허용해야 한다 | 로그인 |
| 13 | 시스템은 인증 실패 시 사용자에게 실패 메시지를 표시해야 한다 | 로그인 |
| **14 | 관리자는 주어진 ID(admin)와 비밀번호(admin)로 로그인할 수 있다 | 로그인 |
| 15 | 시스템은 로그인된 사용자가 로그아웃 버튼을 클릭하면 세션을 종료해야 한다 | 로그아웃 |
| 16 | 관리자는 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 입력하여 설문을 등록할 수 있다 | 설문 등록 |
| 17 | 관리자는 등록된 모든 설문 리스트를 조회할 수 있다 | 설문 리스트 조회 |
| 18 | 관리자는 설문 리스트 조회 화면에서 원하는 설문 항목을 선택하여 상세 내용을 팝업 창으로 볼 수 있다 | 설문 상세 팝업 조회 |
| 19 | 설문 상세 팝업 창에서 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 표시해야 한다 | 설문 상세 팝업 조회 |
| 20 | 관리자는 설문 리스트 조회 화면에서 특정 설문 항목을 선택하여 삭제할 수 있다 | 설문 삭제 |
| 21 | 회원이 검색 대상 설문의 진행 상태(진행 중 / 진행 예정 / 종료)를 선택할 수 있도록 한다 | 설문 검색 |
| 22 | 회원이 검색 키워드를 입력할 수 있도록 한다 | 설문 검색 |
| 23 | 시스템은 입력한 키워드와 선택한 진행 상태 조건에 부합하는 설문 리스트를 출력한다 | 설문 검색 |
| 24 | 회원이 검색 결과 리스트에서 특정 설문을 선택할 수 있도록 한다 | 설문 검색 |
| 25 | 조건에 맞는 설문이 없는 경우 빈 리스트 또는 결과 없음을 표시한다 | 설문 검색 |
| **26 | 검색 기능은 로그인된 회원만 사용할 수 있도록 제한한다 | 로그인된 회원이 수행하는 모든 use case(공통 요구사항) |
| 27 | 검색된 설문 리스트에서 특정 설문을 선택할 수 있다 | 설문 상세 정보 조회 |
| 28 | 특정 설문 선택 시 새로운 화면에서 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각이 출력된다 | 설문 상세 정보 조회 |
| 29 | 현재 진행 중인 설문인 경우 상세 정보 화면에서 응답 버튼이 활성화된다 | 설문 상세 정보 조회 |
| 30 | 활성화된 응답 버튼을 눌러 응답 화면으로 이동할 수 있다 | 설문 응답 |
| 31 | 응답 화면에서 설문에 응답하고 응답 버튼을 눌러 제출할 수 있다 | 설문 응답 |
| 32 | 자신이 응답한 모든 설문 정보를 한 화면에서 조회할 수 있다 | 응답한 설문 조회 |
| 33 | 응답한 설문 조회 화면에서 각 설문에 대해 설문 제목, 응답 내용, 응답 일시가 출력된다 | 응답한 설문 조회 |
| 34 | 현재 진행 중인 설문에 한하여 제출한 응답을 수정할 수 있다 | 응답 수정 |
| 35 | 응답 수정 화면에는 가장 최근에 수정한 날짜가 출력된다 | 응답 수정 |
| 36 | 수정 버튼을 눌러 응답 수정을 완료할 수 있다 | 응답 수정 |
| 37 | 설문이 종료된 경우 응답을 수정할 수 없다 | 응답 수정 |
| 38 | 현재 진행 중인 설문에 한하여 응답을 취소할 수 있다 | 응답 취소 |
| 39 | 설문이 종료된 경우 응답을 취소할 수 없다 | 응답 취소 |
| 40 | 시스템은 관리자에게만 설문 응답 통계 조회 기능을 제공해야 한다 | 통계 |
| 41 | 시스템은 통계 조회 시 검색 단위를 최근 1주일 / 1개월 / 1년 중 선택할 수 있는 옵션을 제공해야 한다 | 통계 |
| 42 | 시스템은 선택된 기간을 기준으로 전체 설문 수를 산출하여 출력해야 한다 | 통계 |
| 43 | 시스템은 선택된 기간을 기준으로 종료된 설문 수를 산출하여 출력해야 한다 | 통계 |
| 44 | 시스템은 선택된 기간을 기준으로 총 응답 수를 산출하여 출력해야 한다 | 통계 |
