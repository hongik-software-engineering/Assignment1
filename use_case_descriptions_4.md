## Use Case Description

| Use Case | Description |
| --- | --- |
| 설문 상세정보 조회 | 회원이 검색된 설문 리스트에서 특정 설문을 선택하면 새로운 화면에서 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각이 출력된다. 현재 진행 중인 설문인 경우 응답 버튼이 활성화된다. |
| 응답 버튼 활성화 | 설문 상세정보 조회 화면에서 현재 진행 중인 설문인 경우에 한하여 응답 버튼이 활성화된다. |
| 설문 응답 | 회원이 활성화된 응답 버튼을 눌러 응답 화면으로 이동한다. 회원은 설문 문항에 응답을 입력하고 응답 버튼을 눌러 제출한다. |
| 응답한 설문 조회 | 회원이 자신이 응답한 모든 설문 정보를 한 화면에서 조회할 수 있다. 각 설문에 대해 설문 제목, 응답 내용, 응답 일시가 출력된다. |
| 응답 수정 | 회원이 현재 진행 중인 설문에 한하여 응답을 수정할 수 있다. 수정 화면에는 가장 최근에 수정한 날짜가 출력되며 수정 버튼을 눌러 완료한다. 설문이 종료된 경우 수정이 불가하다. |
| 응답 취소 | 회원이 현재 진행 중인 설문에 한하여 응답을 취소할 수 있다. 설문이 종료된 경우 취소가 불가하다. |

#### 1. 설문 상세 정보 조회

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 검색된 설문 리스트 화면에서 특정 설문을 선택한다. |  |
| 2 |  | 새로운 화면에서 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 출력한다 |
| 3 |  | 현재 진행 중인 설문인 경우 응답 버튼을 활성화한다 |
|  |  |  |

### 2. 설문 상세정보 조회

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 검색된 설문 리스트에서 특정 설문을 선택한다 |  |
| 2 |  | 시스템은 새로운 화면에서 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 출력한다 |
| 3 |  | 시스템은 현재 진행 중인 설문인 경우 응답 버튼을 활성화한다 |

---

### 3. 설문 응답

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 활성화된 응답 버튼을 누른다 |  |
| 2 |  | 시스템은 응답 화면으로 이동한다 |
| 3 | 회원은 설문 문항에 응답을 입력한다 |  |
| 4 | 회원은 응답 버튼을 누른다 |  |
| 5 |  | 시스템은 응답을 저장하고 완료 처리한다 |

---

### 4. 응답한 설문 조회

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 응답한 설문 조회 화면으로 이동한다 |  |
| 2 |  | 시스템은 해당 회원이 응답한 모든 설문 정보를 한 화면에 출력한다 |
| 3 |  | 시스템은 각 설문에 대해 설문 제목, 응답 내용, 응답 일시를 출력한다 |

---

### 5. 응답 수정

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 응답한 설문 조회 화면에서 수정할 설문의 응답 수정을 선택한다 |  |
| 2 |  | 시스템은 응답 수정 화면으로 이동하며 가장 최근에 수정한 날짜를 출력한다 |
| 3 | 회원은 응답 내용을 수정한다 |  |
| 4 | 회원은 수정 버튼을 누른다 |  |
| 5 |  | 시스템은 수정된 응답을 저장한다 |

예외: 설문이 종료된 경우 수정 버튼이 비활성화되어 응답 수정이 불가하다

### 6. 응답 취소

| Step | User Action | System Response |
| --- | --- | --- |
| 1 | 회원은 응답한 설문 조회 화면에서 취소할 설문의 응답 취소를 선택한다 |  |
| 2 |  | 시스템은 취소 확인 메시지를 표시한다 |
| 3 | 회원은 확인 버튼을 누른다 |  |
| 4 |  | 시스템은 해당 응답을 삭제한다 |
| 5 |  | 시스템은 갱신된 응답한 설문 조회 화면을 표시한다 |

예외: 설문이 종료된 경우 응답 취소가 불가하다.
