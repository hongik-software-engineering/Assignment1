### 설문 상세 정보 조회 및 응답 기능
### 응답한 설문 조회/수정/취소 기능

| No. | Requirement | Use Case(s) |
| --- | --- | --- |
| 1 | 검색된 설문 리스트에서 특정 설문을 선택할 수 있다 | 설문 상세 정보 조회 |
| 2 | 특정 설문 선택 시 새로운 화면에서 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각이 출력된다 | 설문 상세 정보 조회 |
| 3 | 현재 진행 중인 설문인 경우 상세 정보 화면에서 응답 버튼이 활성화된다 | 설문 상세 정보 조회 |
| 4 | 활성화된 응답 버튼을 눌러 응답 화면으로 이동할 수 있다 | 설문 응답 |
| 5 | 응답 화면에서 설문에 응답하고 응답 버튼을 눌러 제출할 수 있다 | 설문 응답 |
| 6 | 자신이 응답한 모든 설문 정보를 한 화면에서 조회할 수 있다 | 응답한 설문 조회 |
| 7 | 응답한 설문 조회 화면에서 각 설문에 대해 설문 제목, 응답 내용, 응답 일시가 출력된다 | 응답한 설문 조회 |
| 8 | 현재 진행 중인 설문에 한하여 제출한 응답을 수정할 수 있다 | 응답 수정 |
| 9 | 응답 수정 화면에는 가장 최근에 수정한 날짜가 출력된다 | 응답 수정 |
| 10 | 수정 버튼을 눌러 응답 수정을 완료할 수 있다 | 응답 수정 |
| 11 | 설문이 종료된 경우 응답을 수정할 수 없다 | 응답 수정 |
| 12 | 현재 진행 중인 설문에 한하여 응답을 취소할 수 있다| 응답 취소 |
| 13 | 설문이 종료된 경우 응답을 취소할 수 없다 | 응답 취소 |
