# requirement list

**회원 탈퇴**

| **No.** | **Requirement** | **Use Case(s)** |
| --- | --- | --- |
| 1 | 회원은 시스템에서 언제든 탈퇴할 수 있도록 한다 | 회원 탈퇴 |
| 2 | 탈퇴 전에 본인 확인을 위해 비밀번호를 재입력하도록 한다 | 회원 탈퇴 |
| 3 | 탈퇴 전에 최종 확인 단계를 거치도록 한다 | 회원 탈퇴 |
| 4 | 탈퇴 시 시스템은 회원 정보(ID, 비밀번호, 이름, 전화번호, 이메일 등)를 삭제한다 | 회원 탈퇴 |
| 5 | 탈퇴 시 회원이 제출한 설문 응답 데이터를 삭제한다 | 회원 탈퇴 |
| 6 | 탈퇴 후에는 해당 ID로 시스템 이용이 불가능하도록 모든 이용 권한을 회수한다 | 회원 탈퇴 |
| 7 | 탈퇴 처리가 완료되면 자동으로 로그아웃 처리한다 | 회원 탈퇴 |
| 8 | 탈퇴 완료 메시지를 회원에게 표시한다 | 회원 탈퇴 |
| **9 | 탈퇴는 로그인 된 회원만 수행할 수 있도록 제한한다 | 로그인 된 회원이 수행하는 모든 use case(공통 요구사항) |

**설문 검색**

| **No.** | **Requirement** | **Use Case(s)** |
| --- | --- | --- |
| 1 | 회원이 검색 대상 설문의 진행 상태(진행 중 / 진행 예정 / 종료)를 선택할 수 있도록 한다 | 설문 검색 |
| 2 | 회원이 검색 키워드를 입력할 수 있도록 한다 | 설문 검색 |
| 3 | 시스템은 입력한 키워드와 선택한 진행 상태 조건에 부합하는 설문 리스트를 출력한다 | 설문 검색 |
| 4 | 회원이 검색 결과 리스트에서 특정 설문을 선택할 수 있도록 한다 | 설문 검색 |
| 5 | 조건에 맞는 설문이 없는 경우 빈 리스트 또는 결과 없음을 표시한다 | 설문 검색 |
| **6 | 검색 기능은 로그인된 회원만 사용할 수 있도록 제한한다 | 로그인된 회원이 수행하는 모든 use case(공통 요구사항) |
